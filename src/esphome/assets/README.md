# Display Asset Notes

Place the face assets here as:

- `HAL_200x200.png`
- `ALIEN_200x200.png`

Recommended asset format:

- transparent PNG
- `200x200`
- centered subject with transparent padding if needed

Theme selection:

- set the active theme in `../face-theme-params.yaml`
- future frame arrays belong under `face-arrays/`

Why not full-screen?

A smaller centered transparent image is more memory-efficient than a full
`320x240` full-screen image and still gives the requested single pulsing
center-screen visual.
